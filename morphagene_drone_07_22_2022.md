# Morphagene Drone Patch 07/24/2022

## Modules Used:

* Morphagene
* SSF Dipole Filter
* Serge Resonant EQ
* ALM MFX
* Mannequins Cold Mac
* NLC VCAs
* NLC Clump Matrix Mixer
* Music thing Modular Startup
* Montreal Assembly CT5

------

<br>

## Settings

<br>

### Morphagene

* Norns Drone sample, mono out to matrix mixer CH 1
* Morph full CCW
* Octave down playback

### SSF Dipole

* Stereo inputs from MFX
* Dipole outs to MTM startup
* Pole B output to Res EQ Ch2 in
* Dipole AB to ST-fm in (feedback)

### Resonant EQ
* Out >>> Vca Ch1 in
* Configure EQ to simulate kick drum

### ALM MFX
* Input from matrix mixer
* Almicon reverb, to taste
* Stereo outs to dipole A/B in

### MTM startup
* Audio mix from SSF dipole
* Diviply out to Res EQ CH 1 in

### Cold Mac
* Location to dipole stereo 1v/oct

### NLC VCAs
* Input Ch1 from Res EQ
* Output Ch1 to Clump Ch3

### NLC Clump

* Input 1, Morphagene Playback
* Input 2, Ct5
* Input 3, Res EQ Combined out (From Vca)

* Output 1 >>> Ct5
* Output 2 >>> ALM MFX input
* Output 3 >>> Dry output to MTM Startup

### MTL/ASM Ct5

* Mode 1 
* low/medium feedback 
* moderately long buffer time (to taste)
* Fifths up quantization
